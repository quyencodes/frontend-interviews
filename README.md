## Project Description

Each folder represents a component built to practice fundamental front-end skills. Click a folder and follow `Getting Started` to see it on your local browser.

## Getting Started

1. Fork and clone the repository:
   ```bash
   $ git clone https://github.com/quyencodes/frontend-interviews.git
   ```
2. Open the terminal with `ctrl + backtick mark`, make sure to <b>change to the correct directory</b> and write the following command (install NPM packages):
   ```bash
   npm install
   ```
3. Open a seperate terminal window and write the following command:
   ```bash
   npm run dev
   ```
4. Project will be deployed on a localhost server, navigate to your browser and enter in the url:
   ```bash
   http://localhost:3000
   ```
